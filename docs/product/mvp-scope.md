# MVP Scope

## Objetivo
Permitir a un usuario registrar y consultar sus mediciones de tensión arterial en modo anónimo usando Local Storage.

## Incluido
- Registro manual
- Registro por foto (OCR)
- Persistencia en Local Storage
- Listado de mediciones
- Arquitectura separada frontend/backend
- Tests básicos

## Excluido
- Login
- Google Drive
- Recordatorios
- Gráficas
- Diagnóstico médico
