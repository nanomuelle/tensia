# User Stories

## US-01 Registro manual
Como usuario
Quiero registrar mi tensión manualmente
Para llevar control personal

### Aceptación
- Sistólica y diastólica obligatorias
- Persistencia tras recargar
- Confirmación visual

---

## US-02 Registro por foto
Como usuario
Quiero subir una foto del tensiómetro
Para no introducir datos manualmente

### Aceptación
- OCR extrae valores
- Usuario puede editar antes de guardar
